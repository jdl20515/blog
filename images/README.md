# Images Directory

Place all your blog post images in this directory.

## Usage

Reference images in your blog posts like this:

```html
<img src="../images/your-image.jpg" alt="Description">
```

## Organization Tips

- Use descriptive filenames (e.g., `sunset-over-mountains.jpg` instead of `img1.jpg`)
- Keep file sizes reasonable for web use
- Common formats: JPG, PNG, GIF, WebP
