# Minimalist Blog Template

An extremely minimalist blog template built with base HTML and minimal CSS.

## Features

- Clean, simple design
- Responsive layout
- No dependencies
- Easy to customize

## Usage

1. Open `index.html` in your browser
2. Edit the HTML files to add your own posts
3. Customize the minimal CSS as needed

## Structure

- `index.html` - Main blog page with post listings
- `post.html` - Individual post template
